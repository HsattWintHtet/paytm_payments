# paytm_payments

Paytm Payments Flutter SDK

## Usage

[Example](https://github.com/apoorv9496/paytm_payments/blob/master/example/lib/main.dart)

To use this package :

* add the dependency to your pubspec.yaml file.

```yaml
  dependencies:
    flutter:
      sdk: flutter
    paytm_payments:
```
